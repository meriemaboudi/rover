# Rover

## Prerequisites
In order to build and run the project, you need
- JDK 21 or higher
- Maven

## Project Build
To build the project you simply need to run the following command
```shell
mvn clean package
```

## Project Run
In order to run the application with an input file (for instance `data/rover.txt`), you can use the following command
```shell
java -jar target/rover.jar data/rover.txt
```
